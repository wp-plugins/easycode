== Description ==

<strong>A great tool for programmers' blog. <br/>
Insert programming code in your posts.</strong>
<br/>
<p>
With the help of the plugin you can share your programming code in a better way. The plugin provides your website with a more friendly and more convenient interface with the abilities to collapse code areas, copy code and edit code.
 </p>
<br/>
<p>The latest version is 1.2. There are some new features :
</p>
<ul>
<li>Customizing the colors and size </li>
<li>Allow using simple HTML tags. </li>
<li>Analyze code smartlier.</li>
</ul>
<p></p>
<p><strong>GUIDANCE: How to insert code in your posts?</strong></p>
<p>Place use <i>[code]бн[/code]</i> :just put your code between them. You are allowed to use simple HTML tags such as <i>&lt;strong&gt;&lt;i&gt;&lt;span&gt;&lt;u&gt;&lt;big&gt;&lt;small&gt;</i></p>

<p>Known Errors: Some browsers cannot show the interfaces correctly when the option <i>word-wrap:break-word</i> is enable. & (Edit mode) Some older browsers output all of the code in one line.
