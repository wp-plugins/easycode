== Description ==

**A great tool for programmers' blog. Insert programming code in your posts.**

With the help of the plugin, you can share your programming code in a better way. The plugin provides your websites with a more friendly and more convenient interface.

How to use the plugin?Please refer to the 'Frequently Asked Questions' section.

== Installation ==

1. Upload the `easycode` directory in the zip file to the `/wp-content/plugins/` directory in your server.
1. Activate the plugin through the 'Plugins' menu in WordPress

*It's also OK to install by searching the online plugin library in the 'Plugins' menu.

== Frequently Asked Questions ==

= How to use EasyCode in my posts? =
1.When writing your posts,please click the blue EasyCode button on the toolbar.
1.Input code you want to insert.
1.If you want to attach a title to you code,please input it.
1.Click 'OK' to insert your code.

= Which tag should I choose? =

In EasyCode,there are three tags:
1.[code]...[/code]
1.<code></code>
1.<pre></pre>

*The default tag <pre> is recommended.*
In some browsers,the tabs and space may be ignored by the visual editor when using tags [code] or <code>.


= Can the plugin handle comments? =

Of course! Once the plugin is activated,it will add 'Insert Code' button before the comment input field.It can also handle code in comments just like in posts.

If you don't want to enable the function,please adjust in admin page.


= Where is the option page? =

It is in `Setting` menu.

= Can I use my own color to suit my blog themes? =

Naturally.
In 'Appearance' Tag of the option page, you can select the color schemes offered by EasyCode.
If you are familiar with the computer color,you can also choose your own colors.

*All the CSS files for color schemes are located in `/boxstyles`. *
