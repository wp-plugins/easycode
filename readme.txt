== Description ==

**A great tool for programmers' blog. 
Insert programming code in your posts.**

With the help of the plugin you can share your programming code in a better way. The plugin provides your website with a more friendly and more convenient interface with the abilities to collapse code areas, copy code and edit code.

The latest version is 1.2. There are some new features :
* Customizing the colors and size
* Allow using simple HTML tags.
* Analyze code smartlier.

**GUIDANCE: How to insert code in your posts?**
Place use <i>[code]бн[/code]</i> :just put your code between them. You are allowed to use simple HTML tags such as <i>&lt;strong&gt;&lt;i&gt;&lt;span&gt;&lt;u&gt;&lt;big&gt;&lt;small&gt;</i>
